# AccessibleFonts Credits

This package includes the following open source fonts. We are grateful to the
designers and organizations that have made these fonts available.

---

## OpenDyslexic

**Author:** Abbie Gonzalez  
**Website:** https://opendyslexic.org  
**License:** SIL Open Font License 1.1  

OpenDyslexic is a typeface designed against some common symptoms of dyslexia.
The typeface uses heavy weighted bottoms to indicate direction and prevent
letter rotation and flipping.

**Attribution:**  
OpenDyslexic © 2011-2019 Abbie Gonzalez (https://opendyslexic.org)

---

## Atkinson Hyperlegible

**Author:** Braille Institute of America, Inc.  
**Website:** https://brailleinstitute.org/freefont  
**License:** SIL Open Font License 1.1  

Named after Braille Institute founder J. Robert Atkinson, Atkinson Hyperlegible
font focuses on letterform distinction to increase character recognition,
ultimately improving readability. The font was designed by Applied Design Works.

**Attribution:**  
Atkinson Hyperlegible © 2020 Braille Institute of America, Inc.

---

## Lexend

**Authors:** Thomas Jockin, Bonnie Shaver-Troup EdD  
**Website:** https://www.lexend.com  
**License:** SIL Open Font License 1.1  

Lexend is a variable font family based on original research by Dr. Bonnie
Shaver-Troup which explores how certain font characteristics can improve
reading fluency.

**Attribution:**  
Lexend © 2018 The Lexend Project Authors (https://github.com/googlefonts/lexend)

---

## Inter

**Author:** Rasmus Andersson  
**Website:** https://rsms.me/inter/  
**License:** SIL Open Font License 1.1  

Inter is a typeface carefully crafted & designed for computer screens. Inter
features a tall x-height to aid in readability of mixed-case and lower-case text.

**Attribution:**  
Inter © 2016 The Inter Project Authors (https://github.com/rsms/inter)

---

## Open Sans

**Author:** Steve Matteson  
**Website:** https://fonts.google.com/specimen/Open+Sans  
**License:** SIL Open Font License 1.1  

Open Sans is a humanist sans serif typeface designed by Steve Matteson.
Open Sans was designed with an upright stress, open forms and a neutral,
yet friendly appearance.

**Attribution:**  
Open Sans © 2020 The Open Sans Project Authors (https://github.com/googlefonts/opensans)

---

## Inconsolata

**Author:** Raph Levien  
**Website:** https://levien.com/type/myfonts/inconsolata.html  
**License:** SIL Open Font License 1.1  

Inconsolata is an open-source font created by Raph Levien and released under
the SIL Open Font License. It is a humanist monospaced font designed for
source code listing, terminal emulators, and similar uses.

**Attribution:**  
Inconsolata © 2020 Raph Levien
